这里收集的是目前最完整的Unicode国际字符表清单

需要注意的明，Unicode字符集并不是一成不变的，有不同的制订标准和版本迭代，同时作为技术有一定的延迟性（这也包括国标），所以有时会出现国际码总位数在不同的版本标准上不同的现象，我们尽量保持了完整的目录，这缘于逐浪字库研发的需要（有的人把字库开发当作一个图形设计工作，有的人把字库研发当作一个创意工作 ，有的人把字库工作当作一个概念炒作，但我们认为这是一个纯粹的编码研发工程体系，需要严谨面对，不止是倒腾图形工具这么简单）。

整个编码的描述，我们尽量提供足够多的背景，同时开放给业界同行，汲取中日韩以及繁体汉字人群的使用习惯，进行完整的描述。

在HTML中，可以采用如 &#10084;输出一个❤符号，这也是网页中常用符号的方法。

在网页中，有时候我们会需要一些符号，例如：倒三角 ▼。如果要让网页显示这种符号，显然用字符的方式比较好，体积小到可以忽略不计同时还可以方便用 CSS 定义样式等。但是直接把这个倒三角复制进网页中的话，虽然可以，但是不合理，这时候我们就要用到对应图标的实体字符，例如通过查询可以看到倒三角对应的 Unicode HTML 代码为 &#9660;，复制进 HTML 文件中就可以看到了。

在CSS中，则可以采用 .class::before { content: '\2689'; ... 方法输出一个⚉符号。

由于涉及二进制转码问题，这样的原生符号输出，仅支持数字开始的码，如274E（Dingbats符号集符号集即此类型）。

目前业界对于图标解决方案则是生成一套专用字体来输出，比如Font Awesome奥林图标以及逐浪软件的Afont系列。
